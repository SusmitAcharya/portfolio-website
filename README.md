# Susmit Acharya — Personal Portfolio

Welcome. This repository contains the source code for my personal portfolio website: [www.susmitacharya.com](https://www.susmitacharya.com).  
It’s built to showcase my background, skills, projects and academic/research interests.

---

## Overview

This portfolio site reflects my journey as a student, coder, researcher, and developer. I created it to:  
- Provide a clean, professional, centralized presence online.  
- Highlight my skills, projects, and research work.  
- Make it easier for collaborators, recruiters, or peers to get to know my profile without digging through scattered links.

What this site offers:  
- About / Bio section along with my resume 
- Skills & technology stack overview  
- Projects / Work section (with links, descriptions, maybe live demos) with detailed case studies for my major projects.
- Contact / Social links
